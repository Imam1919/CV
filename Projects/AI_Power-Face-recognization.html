The AI models have been separated for each one of them to operate and generate first then then go to the other one. There is 4 AI deduction models in the project:

1. Face deduction model: this model have to be the first one in the queue because all the other need the face deduction to send them data about the face so they can start to operate and here is how the model is working. once it get the picture that need to be processed it will search for the human eyes, because is one of the easiest features to deduct. then it will try to detect facial landmarks, such as eyebrows, mouth, nose, nostrils and irises. Once the algorithm concludes that it has found a facial region, it does additional tests to confirm that it has detected a face. There are deferent methods to train the program to deduct faces the one that have been used in the project is the: Feature-based or feature-invariant: This method uses features such as a person's eyes or nose to detect a face. They can be negatively affected by noise and light. the face will be deducted then the python code will do a box shape line around the face to show that it has been deducted.

2. Gender and age deduction model: UTKFace dataset is a large-scale face dataset with long age span (range from 0 to 116 years old). The dataset consists of over 20,000 face images with annotations of age, gender, and ethnicity. The images cover large variation in pose, facial expression, illumination, occlusion, resolution, etc. This dataset could be used on a variety of tasks, e.g., face detection, age estimation, age progression/regression, landmark localization, etc. the issue with this data set that it need the face to be supper clear in the frame so that it can give a correct answer, if the face is in a side or it deducted in a video where the face is moving you will find that the age and gender is changing based on the movement of the face. That means it can not give a correct answer all the time. The data set for the age and gender also focuses in the wrinkles of the face where it can read the older people by there wrinkles of the face and the teenagers by the shape of there faces while it will deduct the babies by the roundness of there faces. The less that the ai sees wrinkles the less the age that it will deduct. Not like the gender deduction where the gender deduction will start with palate where In the female the palate will be smaller than the palate in the males same goes for the forehead where the males usually have a squared shaped forehead while the girls will have it more to rounded.

4. race deduction model. As a data set for this model I have used deepface library to deduct the race of the target. Where deepface race deduction model actually CNN models and they expect standard sized inputs. So, resizing is required before representation. To avoid deformation, deepface adds black padding pixels according to the target size argument after detection and alignment.
