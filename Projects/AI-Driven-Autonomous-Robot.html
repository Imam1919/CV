AI-Driven Autonomous Robot

Description:

This project presents an AI-controlled autonomous robot designed to intelligently track and follow a specific person or object while carrying items and avoiding obstacles in real time. The system can be used as a smart shopping assistant, farming support robot, or logistics assistant for domestic, semi-commercial, and commercial environments.

The robot combines computer vision, AI decision-making, and multi-sensor navigation to support autonomous operation. A camera module provides real-time target identification and tracking using OpenCV and TensorFlow Lite, while proximity sensors and LiDAR help scan the environment for obstacle detection and avoidance. Motion control and data processing run on a dedicated embedded computer for reliable robot control.

The system also includes an intuitive mobile application for remote monitoring and control. With integrated carrying space for groceries, agricultural inputs, or industrial tools, the robot demonstrates a practical approach to improving productivity and operational efficiency through AI, sensor fusion, and human-machine interaction.
