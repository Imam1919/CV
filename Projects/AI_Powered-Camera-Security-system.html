AI Powered Camera Security System CCTV

Description:

This project is an AI-powered security and surveillance system that detects and tracks individuals within an organization in real time. It uses computer vision and deep learning to verify if a person follows the required dress code, identify faces, and retrieve stored details from a database. Once identified, the system continuously monitors the personâ€™s movement inside the premises until they leave. If an individual hides their face, moves suspiciously, or loiters in restricted areas, the system flags them as a potential threat and alerts security in real time. Additionally, the AI can detect theft or unusual object placement, such as items being hidden in clothing or bags, and immediately notify security personnel. A live dashboard allows administrators to track individuals, review incidents, and receive instant alerts via SMS or email. This solution enhances security, prevents unauthorized access, and ensures efficient monitoring without constant human supervision.
